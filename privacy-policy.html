1. Overview

Pancake to CRM is an internal browser extension developed by Vaesa Trading Co., Ltd for use by authorized company staff only. It is not available to the general public.

2. Data We Collect

The extension reads the following data from the Pancake CRM interface (pancake.vn) when a staff member actively initiates a sync:

  • Customer name
  • Phone number
  • Facebook conversation ID
  • Facebook user ID
  • Facebook page link
  • Advertising campaign ID (ads_id)
  • Platform type (Facebook / TikTok)

3. How We Use the Data

Data collected is transmitted solely to Vaesa's internal CRM system (vaesa.soly.com.vn) for the purpose of customer record management. It is never sold, shared, or transmitted to any third party.

4. Data Storage

The extension temporarily caches an ads mapping table in the browser's local storage (chrome.storage.local) to improve performance. This cache contains only ad IDs and their corresponding note codes — no personal customer data is stored locally.

5. Permissions Used

  • activeTab — to read data from the currently open Pancake conversation
  • storage — to cache ads mapping data locally for faster lookups

6. Who Has Access

This extension is restricted to employees of Vaesa Trading Co., Ltd. No data is accessible to external parties.

For any questions regarding this privacy policy, please contact us at vaesa.ltd@gmail.com.
